# Memory Game

## Development
clone the repository 
```
git clone https://github.com/simwipado/memory-game
```
install packages
```
npm i
```
run development sever, hot reloading from src directory
```
npm start
```

## Production build
clone the repository 
```
git clone https://github.com/simwipado/memory-game
```
install production packages
```
npm i --only=prod
```
create production build
```
npm run build
```
